## How to Use

This file logs every resolved brand decision. Purpose:

1. **Prevent re-litigation.** If a decision is here, it was made deliberately. Don't reopen without new evidence.
2. **Preserve rationale.** Future contributors (or future you) need to know WHY, not just what.
3. **Track rejected alternatives.** Knowing what was considered and rejected is as valuable as knowing what was chosen.

To reopen a resolved decision: bring new data or changed circumstances. Reference the BRD number. If both founders agree to revisit, move to `pending-alignment.md` with a new PA number and link back to the original BRD.

## BRD-001: Web Typography

- **Decision:** Instrument Serif (display) + Inter (body), served locally via @font-face
- **Options considered:**
  - Pierson + Biryani (original empaque fonts)
  - Instrument Serif + Inter (Google Fonts, libre)
- **Rationale:** Pierson not available as web font with good rendering. Instrument Serif provides same editorial serif character with better web performance and libre licensing. Inter replaces Biryani for cleaner UI readability. Empaque keeps Pierson + Biryani (already printed).
- **Who decided:** Sam (tech) with founder approval
- **Date:** March 2026
- **Status:** Active
